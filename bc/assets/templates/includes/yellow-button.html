{% comment %} This template renders a yellow button that fills the horizontal space available and takes up to three keyword arguments described below: Parameters: link: The URL that the hyperlink points to. text: Text of the button. size: Sets the padding of the button. Possible values are: sm, md, lg It's advisable to wrap this template with a div tag and use the parent element to handle the sizing and the responsive behavior. Here's an example:
{% include 'includes/yellow-button.html' with link="https://free.law/contact/" text='Contact Us' size="sm" %}
